# Лабораторная работа 2
> Выполнила Султанова Диана СТС-511

## Описание
Изучение и практическое применение методов машинного обучения — k-ближайших соседей (kNN) и линейной регрессии — на примере датасета Palmer Penguins и Diamonds. Задачи включали предобработку данных, визуализацию решающих поверхностей kNN, также исследование влияния количества соседей на качество классификации, предсказания цены бриллиантов и оценку качества моделей.

## Заключение
Оба алгоритма успешно реализованы и протестированы. KNN показал высокую точность классификации видов пингвинов, линейная регрессия предсказывает цену бриллиантов. Полученные результаты подтверждают важность правильной предобработки данных для машинного обучения.
