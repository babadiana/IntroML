# Лабораторная работа 2
> Выполнила Султанова Диана СТС-511

## Описание
Изучение и практическая реализация алгоритмов решающих деревьев и случайного леса для задач классификации и регрессии. Работа включала построение базового решающего дерева с нуля, реализацию критериев информативности (энтропия, индекс Джини, дисперсия), анализ оптимальных разбиений признаков и визуализацию зависимостей критериев качества от порогов разделения.

## Заключение
В ходе работы успешно реализован базовый алгоритм построения решающего дерева, изучены основные критерии разделения данных. Полученные навыки позволяют анализировать и визуализировать процесс выбора признаков для разбиения, что вляяется основой для понимания более сложных ансамблевых методов, таких как случайный лес.
